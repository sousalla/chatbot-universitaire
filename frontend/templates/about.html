À propos d'UniBot

UniBot

Assistant Universitaire Intelligent

Objectif du projet

UniBot est un chatbot intelligent conçu pour assister les étudiants de la Faculté Polydisciplinaire de Béni Mellal. Il utilise l'architecture RAG pour fournir des réponses précises aux questions des étudiants.

Technologies utilisées
  • Python 3.10+
  • Flask (Backend)
  • Bootstrap 5 (Frontend)
  • Sentence-Transformers
  • ChromaDB (Vector DB)
  • Architecture RAG
  • API REST
  • JavaScript
Architecture RAG

📚 Phase d'indexation

Documents → Chunks → Embeddings → ChromaDB

💬 Phase de génération

Question → Embedding → Recherche → Contexte → Réponse
